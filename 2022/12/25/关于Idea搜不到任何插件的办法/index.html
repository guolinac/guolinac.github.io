关于Idea搜不到任何插件的办法

发表于 2022-12-25 更新于 2022-12-15 分类于 IDEA
本文字数： 233 阅读时长 ≈ 1 分钟

1.首先来到以下画面
File->Settings->Plugins->⚙->HTTP Proxy Settings…

2.如图所示，勾选 Auto-detect proxy settings 和 Auto-detect proxy configuration URL ，后面输入框不用填写，然后点击 Check connection

3.在弹出的窗口输入以下地址

https://plugins.jetbrains.com/idea

弹出链接成功即可

一路点ok

重启IDEA即可
